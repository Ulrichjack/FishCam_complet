🎬 PRÉPARATION DE LA VIDÉO LINKEDIN
Puisque ton application est maintenant parfaite, voici le script exact que je te conseille pour ta vidéo de démonstration (à enregistrer avec OBS Studio ou Loom).
Durée cible : 2 à 3 minutes maximum. (Les recruteurs ne regardent jamais plus longtemps).
Le Script (Ce que tu fais à l'écran) :
[0:00 - 0:15] L'Introduction (Le Dashboard)
Action : Connecte-toi avec le compte Patron.
Voix off (ou texte à l'écran) : "Voici FishCam ERP, une solution complète pour les poissonneries. Le tableau de bord offre une vue en temps réel sur la santé financière de l'entreprise."
Action : Montre avec la souris les cartes (Total Dettes, Épargne, Bénéfice).
[0:15 - 0:45] Le Cœur du Métier : Les Dettes (Comptes Courants)
Action : Clique sur le menu "Comptes en dette".
Voix off : "La gestion du crédit est vitale en Afrique. Le système alerte automatiquement quand un client dépasse sa limite autorisée."
Action : Clique sur "Rembourser" sur une carte rouge, tape un montant, et valide. Montre que la jauge diminue instantanément.
[0:45 - 1:15] La Facturation Intelligente
Action : Va dans "Factures" -> "Nouvelle Facture".
Voix off : "La saisie des achats est optimisée. Le système se souvient du dernier prix d'achat et calcule automatiquement le poids total et les marges prévisionnelles."
Action : Ajoute un produit (ex: Carpe). Montre que le prix se remplit tout seul. Change la quantité, montre que le poids total se multiplie.
[1:15 - 1:45] La Clôture et l'Audit Financier
Action : Va dans "Clôture Journalière".
Voix off : "L'algorithme de clôture croise les ventes, les emprunts et les remboursements pour calculer l'écart de caisse exact, protégeant ainsi le gérant contre les vols."
Action : Tape un montant dans la caisse pour montrer l'écart qui passe au vert ou au rouge en temps réel.
[1:45 - 2:15] La Résilience et le Cloud
Action : Va dans "Sauvegarde".
Voix off : "L'application est conçue pour les zones à faible connectivité. Elle fonctionne 100% en local, et synchronise automatiquement les bases de données vers Cloudflare R2 dès qu'une connexion internet est détectée."
Action : Clique sur "Pousser vers le Cloud" pour montrer le succès.
[2:15 - 2:30] Conclusion
Action : Montre rapidement le PDF d'un Bilan Mensuel.
Voix off : "Développé avec Angular 20, Spring Boot 3 et déployé via Docker."
C'est un scénario percutant qui met en valeur tes compétences de développeur Full-Stack et d'architecte. Qu'en penses-tu ?
